🍕 Pizza Sales Dashboard

A data analysis project on pizza sales performance, uncovering sales trends, best-selling products, and revenue patterns using Python, SQL, and Power BI.

🎯 Objective

Business problem :
"Analyze pizza sales data to understand revenue trends, top-performing products, peak order times, and customer ordering patterns to support business decisions."

🧰 Tools & Technologies

Python : Exploratory Data Analysis (EDA), data cleaning, and preprocessing
SQL : Querying sales data (revenue, orders, product performance)
Power BI : Interactive dashboard and visualizations
Microsoft Word : Final written report summarizing findings


🔄 Workflow

Data Cleaning & EDA (Python) : Cleaned raw order data, checked for duplicates/missing values, and explored sales trends using pandas/matplotlib/seaborn.
Querying (SQL) : Wrote queries to calculate total revenue, best-selling pizzas, order counts by time/day, and category-wise performance.
Visualization (Power BI) : Built an interactive dashboard with KPIs like total revenue, total orders, average order value, and top-selling pizzas.
Reporting (Word) : Documented the full analysis, methodology, and key business takeaways.


📊 Key Insights

Identified best-selling pizza category and size
Found peak order hours/days of the week
Revenue contribution breakdown by pizza type
